# Example Copilot Prompts

When you need help during debugging, here are good prompts to ask Copilot Chat. These prompts ask for **learning and guidance**, not solutions.

---

## General Debugging Questions

- "What does this error message mean: `[paste error]`?"
- "How do I open the browser Developer Tools to inspect my HTML?"
- "What's the difference between an HTML element and an HTML tag?"
- "Why would a page break if I forget a closing tag?"

---

## HTML & Accessibility

- "How do I make images accessible to screen readers?"
- "What should alt text for an image describe?"
- "What's the correct syntax for an image tag with alt text?"
- "How do I check if my HTML is valid?"
- "What does semantic mean?"
- "What's semantic HTML and why does it matter?"
- "What are the common self-closing tags in HTML?"

---

## CSS Properties & Styling

- "What are the CSS text styling properties? (color, size, font, etc.)"
- "What's the difference between `margin` and `padding`?"
- "How does `display: flex;` work? When should I use it?"
- "What does `flex-direction` do in flexbox?"
- "How do I center text on a page?"
- "What's the correct syntax for a CSS color value?"
- "Why isn't my CSS showing up on the page? What could I check?"

---

## Flexbox Layout

- "What does `justify-content` do in flexbox?"
- "What's the difference between `justify-content` and `align-items`?"
- "How do I make items wrap to the next line in flexbox?"
- "What does `flex-wrap: wrap` do?"

---

## Breaking Down Complex Tasks

- "Look at [Lesson 7 in the README](README.md). What are the main steps I need to complete, in order?"
- "What do I need to build first: the HTML files or the CSS file?"
- "What should I do before I start styling my page?"

---

## File & Path Questions

- "How do I know if my image file path is correct?"
- "Where should image files be stored in my project?"
- "What's the difference between `photo.jpg` and `photo.JPG`?"
- "How do I reference a file in a different folder using a relative path?"

---

## Checking Your Work

- "How do I know if my code is correct?"
- "What does it mean if my page looks different than expected?"
- "How do I compare my page to the expected output?"

---

## Remember

✅ Ask for **explanations, clarifications, and breakdowns**

❌ Don't ask to "fix my code" or "solve this lesson"

The best prompts are **specific and ask for learning**, not solutions!
